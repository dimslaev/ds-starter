#DS Starter 

#### Main tools included: 

- Gulp 4 
- Babel 
- Webpack Stream
- BrowserSync

#### Plugins 

- CSS (less, autoprefixer, clean-css)
- JS (babel, webpack)
- HTML (file-include, html-min, html-replace)
- Static files copied

The env variable is currently set manually in *gulpfile.babel.js*.
